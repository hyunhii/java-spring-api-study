**1주차 요구사항**
- 메뉴 등록
- 메뉴 조회
- 메뉴 수정
- 메뉴 삭제

세부사항
- 메뉴 
  - 이름, 타입, 사용여부로 구성한다.
  - 이름은 공백, null이면 안된다.
  - 이름은 10자까지 입력가능하다.
  - 타입은 음료, 디저트로 나눈다.
  - 사용여부는 Y, N으로 나뉜다.
  - 사용여부의 디폴트 값은 Y 이다.

* 기능
  1. 메뉴 등록
     - 메뉴를 등록할때 이름, 타입을 필수로 넣어야 한다.

  2. 메뉴 조회
     - 메뉴를 모두 조회한다.
     - id 값으로 1건 조회한다.

  3. 메뉴 수정
     - 내용을 변경하고 수정한다.
    
  4. 메뉴 삭제(수정)
     - (유효한 ID값의) 사용여부의 값을 N으로 변경한다.
